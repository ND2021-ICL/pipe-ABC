This folder contains all scripts related to the R analysis of processed pipeline read counts, wrapped up neatly in the Jupyter Notebook.

The Notebook follows the Bash pipeline for converting RNAseq data to read counts.

The following is required for the Notebook to work:

For this notebook, you will need the following dependencies:
- R-Studio/R/R-kernel (if using this Jupyter Notebook)
- DESeq2
- ggplot2
- ggpubr

The Notebook is split into four parts:

- Part 1: Package installation (optional)
- Part 2: Data Preparation
- Part 3: DESeq2 Processing
- Part 4: Data Analysis
